# Трекер технологий

Учебное приложение для отслеживания прогресса изучения технологий в веб-разработке. Позволяет добавлять технологии, устанавливать статус изучения, указывать сроки, категорию, ресурсы и делать заметки.

## Функционал

- Добавление, просмотр и редактирование технологий.
- Категории: Frontend, Backend, Mobile, DevOps, Другое.
- Указание даты начала и дедлайна.
- Заметки и ссылки на ресурсы.
- Массовое редактирование статусов.
- Экспорт данных в JSON.
- Импорт данных из JSON с валидацией.
- Прогресс и статистика.
- Темы оформления (светлая/темная — в разработке).
- Все данные хранятся в `localStorage`.
